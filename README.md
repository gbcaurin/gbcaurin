<h2 style="border-bottom: none; padding-bottom: 0;">Stack e ferramentas</h2>

No front, `React` e `Next.js` com foco em componente limpo e UX funcional. No back, `Node.js` / `Express` para APIs REST e `Python` / `FastAPI` quando o problema pede mais lógica ou integração com IA. Banco: `PostgreSQL` como padrão, `MySQL` quando o ambiente exige. Controle de versão com `Git` e commits no estilo [Conventional Commits](https://www.conventionalcommits.org/). Editor: VS Code — o que importa é não perder tempo com fricção desnecessária.
